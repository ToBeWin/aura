## 💡 提示和建议

### 运行前准备

1. **确保 Ollama 运行**:
   ```bash
   ollama serve
   ```

2. **确保模型已下载**:
   ```bash
   ollama pull qwen3.5:2b
   ```

3. **安装依赖**:
   ```bash
   cd aura
   npm install
   ```

### 常见问题

**Q: 应用启动失败？**
- 检查 Ollama 是否运行（`curl http://localhost:11434`）
- 检查模型是否下载（`ollama list`）

**Q: 编译错误？**
- 运行 `cargo clean` 清理缓存
- 重新运行 `cargo build`

**Q: 前端无法连接后端？**
- 检查 Tauri 是否正确启动
- 查看控制台错误信息
